# Changelog

## [0.5.0] - 2026-02-22

### Added
- Initial Unity SDK release
- WebSocket client with auto-reconnect
- GestureManager singleton MonoBehaviour
- GestureBinding system for Inspector-driven gesture mapping
- Hand visualization (spheres + bones)
- Native C# gesture classifier (no server required)
- Feature extraction (81-dim vector) ported to C#
- Custom editor for GestureManager
- Custom property drawer for gesture bindings
- GestureConfig ScriptableObject
- Sample: BasicGestures (logging)
- Sample: GesturePlayerController (character movement)
- Sample: HandVizDemo (3D hand visualization)
