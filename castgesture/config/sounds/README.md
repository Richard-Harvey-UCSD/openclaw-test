# CastGesture Sound Effects

Place `.mp3` files in this directory to use as custom sound effects.

## Usage

1. Drop any `.mp3` file here (e.g., `airhorn.mp3`)
2. Reference it by filename (without extension) in your mappings
3. The sound will be available at `/sounds/airhorn.mp3`

## Built-in Sounds

CastGesture includes these built-in sounds (no files needed):

| Name       | Description                |
|------------|----------------------------|
| pop        | Light pop / confetti burst |
| whoosh     | Swoosh / fire ignition     |
| explosion  | Impact / screen shake      |
| ding       | Notification chime         |
| applause   | Crowd applause             |
| tada       | Celebration fanfare        |

## Recommended Sources

- [Freesound.org](https://freesound.org) — Free CC-licensed sounds
- [Mixkit](https://mixkit.co/free-sound-effects/) — Free sound effects
- [Pixabay](https://pixabay.com/sound-effects/) — Free for commercial use

## Tips

- Keep files under 1MB for fast loading
- Short clips (< 3 seconds) work best
- 44.1kHz MP3 is the recommended format
